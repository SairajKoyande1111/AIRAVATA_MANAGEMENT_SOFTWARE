Build a complete backend (Node.js + Express + MongoDB using Mongoose) for a 5-user internal company management system. 
There are NO ROLES and NO ADMINS. All 5 users have login access and can view all attendance, client, lead, and follow-up data.

====================================
SYSTEM REQUIREMENTS
====================================

AUTHENTICATION:
- Email + password login.
- JWT-based authentication.
- No admin, no roles. Anyone with an account can access and view all data.
- Provide routes:
    POST /auth/register   (Allow registration for 5 internal users)
    POST /auth/login

====================================
ATTENDANCE MODULE
====================================
Daily attendance rules:
- Work session starts at 10:00 AM.
- Lunch break allowed only after 12:00 PM.
- Lunch break max 1 hour.
- Attendance fields:
    clockIn
    breakStart
    breakEnd
    clockOut
    totalWorkMinutes (auto-calculated)
- Prevent invalid conditions:
    - breakStart < 12:00 PM â†’ reject
    - clockOut must be after clockIn
    - totalWorkMinutes = (clockOut - clockIn) - breakDuration
- Attendance schema:
    {
      userId,
      date (YYYY-MM-DD),
      clockIn,
      breakStart,
      breakEnd,
      clockOut,
      totalWorkMinutes,
      createdAt,
      updatedAt
    }

Attendance API endpoints:
- POST /attendance/clockin
- POST /attendance/break/start
- POST /attendance/break/end
- POST /attendance/clockout
- GET /attendance?date=
- GET /attendance/user/:userId
- GET /attendance/summary?date=YYYY-MM-DD (team-wise summary)
